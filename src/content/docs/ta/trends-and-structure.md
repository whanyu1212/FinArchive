**Trend identification** and **support/resistance (S/R)** analysis are the bedrock of technical trading. Before applying any indicator or pattern, you must first answer: *What direction is the market moving, and where are the key levels?*

---

## Defining a Trend

A trend is a **sustained directional bias** in price over time. Dow Theory defines it through the pattern of **swing highs and lows**:

```
  UPTREND:  Higher Highs (HH) + Higher Lows (HL)
  ─────────────────────────────────────────────────────────
  Price
    │                            HH3
    │                   HH2     /
    │          HH1     /   \   /
    │         /   \   / HL2 \_/
    │        / HL1 \_/
    │       /
    └──────────────────────────────────── Time

  DOWNTREND: Lower Highs (LH) + Lower Lows (LL)
  ─────────────────────────────────────────────────────────
  Price
    │  LH1
    │      \   LH2
    │  LL1  \_/    \   LH3
    │               \_/    \
    │          LL2          \__ LL3
    └──────────────────────────────────── Time

  RANGE (Sideways):  No clear pattern of HH/HL or LH/LL
  ─────────────────────────────────────────────────────────
  Price  ─────────────────────────────── Resistance
    │    ╲ /╲ /╲ /╲ /╲ /
    │     ╲/  ╲/  ╲/  ╲/
         ─────────────────────────────── Support
```

### Trend Strength vs. Direction

```
  A trend can be:
  Strong:    Large bodies, consistent HH/HL, pullbacks shallow
  Weak:      Small bodies, erratic swings, deep pullbacks
  Exhausted: Narrowing swings, reduced momentum into a new high/low

  Trend exhaustion signals:
  → Price makes new high but momentum indicator does NOT (divergence)
  → Candle bodies shrink as trend extends
  → Volume declining on trend thrusts
  → Gaps fill quickly (exhaustion gaps)
```

## Trendlines

A **trendline** connects successive swing lows (in an uptrend) or swing highs (in a downtrend). It acts as **dynamic support or resistance**:

```
  UPTREND TRENDLINE (support):
  Price
    │                          /
    │                    ╱────╱
    │              ╱────╱
    │        ╱────╱
    │  ╱────╱
    │ ╱ (trendline — drawn under swing lows)
    └───────────────────────────────────── Time

  Rules:
  → Requires minimum 2 points to draw, confirmed on 3rd touch
  → The more touches, the stronger the trendline
  → Steeper trendlines are more fragile (unsustainable pace)
  → A break below (or above) the trendline signals potential reversal
```

### Channel Lines

A **channel** is formed by drawing a parallel line to the trendline, connecting swing highs (in an uptrend):

```
  Price channel (uptrend):
        ╱───╱ ─ ─ ─ ╱   (channel top — resistance line)
       ╱   ╱   ╲  ╱
      ╱    ─ ─ ─╲╱    (trendline — support)
     ╱

  Trading channels:
  → Buy at trendline support (lower boundary)
  → Take profit at channel resistance (upper boundary)
  → Break above channel: acceleration (buy)
  → Break below trendline: reversal signal (sell)
```

## Support and Resistance

**Support** is a price level where buying interest is strong enough to halt or reverse a decline. **Resistance** is where selling pressure halts or reverses a rally.

```
  Why do S/R levels work?
  ─────────────────────────────────────────────────────────
  1. Memory: Traders remember where price reversed before
  2. Orders: Limit buy orders cluster at support;
             limit sell orders cluster at resistance
  3. Psychology: Round numbers (1.1000, 150.00) attract orders
  4. Institutional levels: Banks place stops/limits at key levels
  ─────────────────────────────────────────────────────────
```

### Types of Support and Resistance

| Type | Example | Strength |
|---|---|---|
| **Swing high/low** | Prior peak or trough | High |
| **Round numbers** | 1.1000, 150.00, $2000 gold | High |
| **Previous range boundary** | Prior consolidation zone | Medium–High |
| **Moving average** | 200-day MA | Dynamic; medium |
| **Fibonacci level** | 61.8% retracement | Medium |
| **Gap fill level** | Price level left by a gap | Medium |
| **VWAP / POC** | Volume-weighted levels | Medium (intraday) |

### Role Reversal (Support → Resistance)

A critical concept: **once a support level breaks, it becomes resistance** (and vice versa):

```
  Prior support (broken) becomes resistance:
  Price
    │ ─────────────── S/R zone
    │  ╲
    │   ╲
    │    ╲──── break below
    │         ╲
    │          ╲───── bounce back to old support
    │               ╲ → now acts as RESISTANCE
    │                ╲
    └───────────────────────────────────── Time

  This "flip" is one of the most reliable patterns in TA
```

## Market Structure

**Market structure** is the arrangement of swing highs and lows across timeframes. Modern traders (especially in Forex) use structure to define:

### Break of Structure (BOS)
A price move that creates a **new swing high in an uptrend** (or new swing low in a downtrend) — confirms trend continuation:

```
  Uptrend BOS:
  Price
    │                        × ← new HH (BOS confirmed)
    │              ×
    │    ×        / \
    │   / \      /   \
    │  /   \    /     \
    └──────────────────────── Time
         ↑        ↑
        HH1       HH2       HH3 (BOS)
```

### Change of Character (ChoCh)
When an uptrend makes a **lower low** for the first time — potential early reversal signal:

```
  Trend change signal:
  Price
    │         ×   ← last HH
    │        / ╲
    │       /   ╲   × ← lower high (LH) — first warning
    │      /     ╲ /
    │     /       × ← lower low (LL = ChoCh)
    └──────────────────────────────────────── Time
                  ↑
            First sign of trend exhaustion
```

## Key S/R Zones in FX Markets

### Psychological / Round Numbers
In FX, the most significant S/R levels tend to be round numbers:

```
  EUR/USD key psychological levels:
  1.0000 (parity)     — extreme psychological level
  1.0500             — major structural support/resistance
  1.1000             — 100-pip round number
  1.1500             — major weekly S/R

  USD/JPY:
  150.00             — intervention risk zone (BoJ)
  145.00             — BoJ intervention 2022 trigger
  100.00             — major psychological (generational support)

  Gold:
  $2,000             — major round number, long-term resistance now support
  $1,800, $2,500     — key round levels
```

### Previous Daily / Weekly Highs and Lows

```
  Previous Week High (PWH) / Previous Week Low (PWL):
  → Strong institutional S/R zones
  → Many algorithms and bank desks use these as stop placement targets
  → A clean break above PWH often triggers stop-driven momentum
```

## Volume at Price

**Volume Profile** shows how much volume traded at each price level — identifying **high-volume nodes (HVN)** and **low-volume nodes (LVN)**:

```
  Volume profile (sideways):
  Price  │ Volume (horizontal bars)
  ───────┼──────────────────────
  1.0950 │ ████
  1.0900 │ ████████████  ← HVN (Point of Control = POC)
  1.0850 │ ████████
  1.0800 │ ████
  1.0750 │ █             ← LVN (low resistance — price moves fast through here)
  1.0700 │ ████

  HVN = area of price acceptance → acts as S/R
  LVN = price moves through quickly → breakout zone
  POC = highest volume price = strongest S/R
```

## The ATR as a Volatility Context Tool

When placing stops around S/R levels, **ATR (Average True Range)** provides context:

```
  Stop placement rule of thumb:
  Stop = S/R level ± (1.0 × ATR)

  Example:
  EUR/USD support: 1.0800
  ATR(14 daily):   0.0060 (60 pips)

  Stop below support: 1.0800 − 0.0060 = 1.0740
  (gives room for normal volatility without being triggered by noise)
```

## Practical S/R Identification Workflow

```
  Step 1: Start on Weekly chart
  → Mark major swing highs/lows of past 2–3 years
  → Mark any significant round numbers or consolidation zones

  Step 2: Move to Daily chart
  → Identify recent trend structure (HH/HL or LH/LL)
  → Mark recent key levels (past 6 months)
  → Note trendlines and channels

  Step 3: Refine on 4H / 1H
  → Identify precise entry zones within S/R areas
  → Look for candlestick confirmation at key levels
  → Place stops using ATR

  Step 4: Assess risk/reward
  → Minimum 1:2 R:R (risk 1 unit to make 2 units)
  → Large liquid S/R levels = more reliable; tighter target
```
